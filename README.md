## Overview

Single-page portfolio built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies, no API keys. Deployed on Cloudflare Pages via GitHub.

Covers everything that is Alli — content creation, AlliOps autonomous agents, Amazon Influencer storefront, hair & beauty, and Wynston the Goldendoodle (Chief Mood Officer).

## Stack

| Layer | Tech |
|---|---|
| Frontend | HTML5 · CSS3 · Vanilla JS |
| Fonts | Playfair Display · DM Sans · Syne (Google Fonts) |
| Hosting | Cloudflare Pages |
| Analytics | Cloudflare Web Analytics |
| Deployment | GitHub → Cloudflare (auto-deploy on push to main) |

No build step. No npm. No webpack. Just push and it deploys.

## Local Development

No setup required. Just open the file:

```bash
git clone https://github.com/createwithalli/createwithalli-site
cd createwithalli-site
open index.html
```

Or drag `index.html` into any browser.

---

## Deployment

This repo is connected to Cloudflare Pages. Any push to `main` triggers an automatic deploy.

```bash
git add .
git commit -m "your update"
git push origin main
```

Live in ~60 seconds.
